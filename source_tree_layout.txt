::

     Top level
        |
        +------programs/        C entry points for programs.
        |                       programs/evilcandy.c is the entry point
        |                       for the EvilCandy interpreter.
        |
        +------src/             Core C sources
        |       |
        |       +---builtin/    C accelerators for library modules
        |       |
        |       +---types/      Callbacks and methods for built-in types
        |
        +------tests/           Test scripts
        |       |
        |       +---c/          C sources needed only by the test programs
        |                       that are writtten in C.
        |
        +------inc/             Main API headers, included by evilcandy.h
        |       |
        |       +---lib/        Headers for code that is semi-independent,
        |       |               and which can with relative ease be
        |       |               ported to a different project.
        |       |
        |       +---internal/   Internal headers
        |       |
        |       +---evilcandy/  Public API headers
        |       |
        |       +---tests/      Headers for tests/c/
        |
        +-------lib/            The main library of scripts that user
        |                       sources can load by default.  These are
        |                       typically installed in some place like
        |                       /usr/local/share/EvilCandy/
        |
        +-------demos/          EvilCandy scripts that demo the code.
        |
        +-------tools/          Source code generators & such.
        |
        +------etc/             Miscellaneous files and programs.
        |       |
        |       +- evilcandy.1  man page for evilcandy.
        |
        +-----Documentation/    Documentation, which uses its own Makefile
                                system (using Sphinx) separately from the
                                auto-conf/auto-make configuration for the
                                rest of the project.
